English

OpenBookLM is an AI workspace for local Vault documents. It supports text-extraction-based search, citation-backed answers, session chat, standalone chatbot mode, reference review, document reading, and document translation into a new app-managed file.

Supported extraction formats include .txt, .md, .markdown, .pdf, .rtf, .docx, .epub, and .hwpx.
