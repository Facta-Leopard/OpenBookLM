English

Email: shteosis@gmail.com

What to include

  • App version, platform, selected language, and provider mode.
  • For Vault Search issues, include whether search was on, which Vault was connected, and short reproduction steps.
  • For document translation issues, include source file type, target language, output format, and the exact error text.

Troubleshooting

  • Reindex the Vault if supported files are not found.
  • Turn Vault Search off for general chatbot behavior.
  • For Apple On-Device / Foundation Models, confirm device, OS, region, language, and Apple settings support the feature.
  • For ChatGPT Browser Sign-In, OpenAI-compatible API, or Local API failures, check credentials, endpoint reachability, and response format.
  • Reconnect the Vault if iPadOS or macOS no longer grants file access after relaunch.

Document translation and files

  • Translation creates a new file and never overwrites the original document.
  • The option screen lets you choose Markdown or plain text output before translation starts.
  • On iPadOS, generated translations may first be saved in the app-managed folder because the system can limit direct writes to external file locations.
  • Apple On-Device / Foundation Models translation is designed to process smaller text segments to reduce context-limit problems.
  • OpenBookLM can index extracted text from .txt, .md, .markdown, .pdf, .rtf, .docx, .epub, and .hwpx. Editing is limited to .txt, .md, and .markdown.
  • PDF, RTF, DOCX, EPUB, and HWPX open as read-only text-focused previews. Original layout, images, complex tables, and scanned PDFs without OCR text may not be preserved or searchable.

Ads and privacy requests

  • The iPad version may show Google AdMob banner ads. The macOS version does not include AdMob.
  • To report an inappropriate ad, email support with a screenshot when possible.
  • Local app data can be cleared from maintenance/reset controls or by deleting the app. For developer-managed support records, email support from the address used for the original request.

  • 문서 번역은 새 파일을 만들며 원본 문서를 덮어쓰지 않습니다.

  • iPadOS에서는 외부 위치 직접 쓰기가 제한될 수 있어 번역본이 먼저 앱 관리 폴더에 저장될 수 있습니다.

  • OpenBookLM은 .txt, .md, .markdown, .pdf, .rtf, .docx, .epub, .hwpx 파일의 추출 텍스트를 인덱싱할 수 있습니다. 편집은 .txt, .md, .markdown만 지원합니다.
  • PDF, RTF, DOCX, EPUB, HWPX는 읽기 전용 텍스트 미리보기로 열립니다. 원본 레이아웃, 이미지, 복잡한 표, OCR 없는 스캔 PDF는 보존 또는 검색이 제한될 수 있습니다.

日本語

お問い合わせ: shteosis@gmail.com

含める情報

  • アプリのバージョン、プラットフォーム、選択中の言語、Provider モード。
  • Vault Search の問題では、検索のオン/オフ、接続した Vault、短い再現手順。
  • 文書翻訳の問題では、元ファイル形式、対象言語、出力形式、正確なエラーテキスト。

確認事項

  • 対応ファイルが見つからない場合は Vault を再インデックスしてください。
  • 通常のチャットボットとして使う場合は Vault Search をオフにしてください。
  • Apple On-Device / Foundation Models は、端末、OS、地域、言語、Apple 設定が対応している必要があります。
  • ChatGPT Browser Sign-In、OpenAI 互換 API、Local API の問題では、認証情報、エンドポイント到達性、レスポンス形式を確認してください。
  • 再起動後にファイルアクセスが失われた場合は、Vault Connection で Vault を再接続してください。

文書翻訳とファイル

  • 翻訳は新しいファイルを作成し、元の文書を上書きしません。
  • 開始前に Markdown またはプレーンテキストなどの出力形式を選べます。
  • iPadOS では外部保存先への直接書き込みが制限される場合があり、翻訳はまずアプリ管理フォルダに保存されることがあります。
  • Apple On-Device / Foundation Models 翻訳は、コンテキスト制限を減らすため小さなテキストセグメントで処理する設計です。
  • .txt, .md, .markdown, .pdf, .rtf, .docx, .epub, .hwpx の抽出テキストをインデックス化できます。編集は .txt, .md, .markdown のみです。
  • PDF、RTF、DOCX、EPUB、HWPX は読み取り専用のテキスト中心プレビューです。元のレイアウト、画像、複雑な表、OCR のないスキャン PDF は保持または検索できない場合があります。

広告とプライバシー依頼

  • iPad 版では Google AdMob バナー広告が表示される場合があります。macOS 版には AdMob は含まれません。
  • 不適切な広告は、可能であればスクリーンショットを添えてサポートへ報告してください。
  • ローカルアプリデータはリセット機能またはアプリ削除で消去できます。開発者管理のサポート記録については、元の問い合わせアドレスから連絡してください。
